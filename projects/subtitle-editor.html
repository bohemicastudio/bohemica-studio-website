Aplikace pro úpravu a tvorbu titulků k videoprezentacím

Popis projektu

Pro významnou produkčně - technickou společnost v oboru konferencí jsme vyvinuli software pro úpravu a vytváření titulků k videím.

  • Staticky generovaná webová aplikace
  • Napojení na API pro získávání a ukládání dat
  • Synchronizace s vloženým přehrávačem
  • Multijazyčnost

Proces

Technické řešení:

  1. Front-end framework: Vue.js
  2. Vzhled: Tailwind CSS
